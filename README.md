# TryHackMe Walkthroughs

This repository contains my personal walkthroughs for TryHackMe rooms.  
It is designed to help me track progress, practice cybersecurity skills, and document solutions in a structured way.

## Structure
- `rooms/` - individual rooms with task walkthroughs and screenshots
- `templates/` - markdown templates for new walkthroughs
- `notes/` - additional notes, commands, and useful resources

## Disclaimer
Flags are sensitive and may be excluded from public repositories. This repo is primarily for learning and personal reference.

## Goals
- Track my progress on TryHackMe rooms
- Improve penetration testing and cybersecurity skills
- Maintain structured notes for review
